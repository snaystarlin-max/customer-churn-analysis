# Customer Churn & Retention Analysis
Data analysis project exploring customer churn drivers and retention strategies using Python

## 📌 Key Insights

- Customers on **month-to-month contracts** have the highest churn rates  
- **Early-stage customers** are significantly more likely to churn  
- Higher **monthly charges** are associated with increased churn  
- Long-term customers show **strong retention and lower churn risk**  

## 🚀 Project Structure

- Data Cleaning & Preparation  
- Exploratory Data Analysis (EDA)  
- Visual Insights  
- Business Recommendations

## 🛠️ Tools Used

Python · Pandas · Matplotlib · Seaborn
